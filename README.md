# 🛒 Amazon Sales Analysis Dashboard

## 📊 Project Overview
This project focuses on analyzing Amazon sales data to generate insights into revenue trends, product performance, and customer behavior. The dashboards were built using Power BI for effective data visualization and decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Understand customer purchasing trends
- Evaluate sales distribution across different regions

---

## 🛠 Tools & Technologies Used
- Power BI (Data Visualization & Dashboard)

---

## 📈 Dashboards Created
1. **Sales Overview Dashboard**
   - Total revenue and sales trends
   - Time-based analysis

2. **Product Performance Dashboard**
   - Top-selling products
   - Category-wise performance

3. **Regional Analysis Dashboard**
   - Region-wise sales distribution
   - Sales comparison across locations

---

## 🔍 Key Insights
- Sales show noticeable trends over time
- A few products contribute significantly to total revenue
- Certain regions outperform others in sales
- Product categories vary in performance and demand

---

## 🚀 Conclusion
This project demonstrates the use of Power BI to transform raw sales data into meaningful insights that support business decision-making.
